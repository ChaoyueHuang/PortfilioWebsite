AUDIO - VISUAL REACTION INSTALLATION

2019.4 - 2019.5
Generative design, Interaction design, VJing

TOOLS:

Max/MSP/Jitter

Overview

It is an audio-visual reaction installation. The participants will put some wood pieces on the top of a top-transparent box, the patterns on the wood pieces are recognized and will trigger the corresponding audio. The visual effect and audio are generated by different wood pieces and their position and angle.

Ideation

At first, I wanted to make an input device that is integrated with several sensors, like the ultrasonic sensor and force-sensitive resistor. However, this kind of interactive method is a little limited and monotonous. Then I came up with the idea of using Kinect to detect gestures. It will be more fun.

In order to create incredible visuals, I learned through the jit.gl family slowly, which contains jit.world, jit.gridshape, jit.gen, jit.mesh, etc. I have mastered how to create basic shapes in the jit.world and apply some changes to it. Besides, I have created a basic audio-visual reactive patch as an experiment.

However, I was not satisfied with the previous design conpect, I would like to explore some new interaction method. Since we can interact with a system by recognizing people's gesture and body movement, why not apply object recognition?
